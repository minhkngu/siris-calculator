# Yêu cầu cập nhật & Nhật ký thay đổi

Thư mục này dùng để lưu trữ các yêu cầu tính năng mới, các bản cập nhật và ghi chú về các thay đổi trong tương lai của dự án **Hotel Price Calculator**.

## 📋 Danh sách yêu cầu chờ xử lý

- [ ] *Ví dụ: Thêm tính năng xuất hóa đơn ra file PDF*
- [ ] *Ví dụ: Tích hợp thanh toán qua QR Code (VNPay/Momo)*
- [ ] *Ví dụ: Thêm giao diện Dark Mode*

## 🔄 Quy trình cập nhật

1. Lưu file yêu cầu mới vào thư mục này (ví dụ: `update-YYYY-MM-DD.md`).
2. Thực hiện cập nhật mã nguồn.
3. Cập nhật trạng thái trong danh sách này.
